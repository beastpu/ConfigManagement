CMDB

开服部署时需要修改以下相关模块:

  • 出档脚本 -- 构建数据档及推送镜像的脚本
  • enter_point脚本 -- 容器启动时加载的脚本
  • script/fixsetting -- MSDK,开关等相关脚本
  • script/config -- dispatch,globallocale的nginx配置脚本

4大模块进行了重构,代码和配置完全分离开来,通过CMDB集中管理配置.功能如下:

  • 可视化编辑
  • API
  • 操作审计
